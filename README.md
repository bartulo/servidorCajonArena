# Instalar

```
git clone https://github.com/bartulo/servidorCajonArena.git
cd servidorCajonArena
npm install
npm run build
```
Cuando termine de ejecutarse
```
Ctrl+C
```

# Crear servidor

```
npm run watch
```
No cierres la terminal y visita:

http://localhost:3000/

Si quieres crear escenarios nuevos:

http://localhost:3000/mapas
